Integers
========

Pushing integers
-----------------

To push a single-digit integer, simply type the number, like so:

::2

To push a multi-digit integer, enclose the number in underscores:

::_57_

Arithmetic
----------

Silicon uses prefix notations, as the integers have to be pushed onto the stack before using an operator.

`_38_4+`

`_44_2-`

`69*`

`_84_2/`

Other
-----

* `^` Exponent
* `!` Factorial
* `q` Square root
* `s` Square
* `r` Random number in the range 1-100

* `F` _n_th Fibonacci number
* `K` Greatest Common Divisor
* `P` Primality checker

* `N` Increment
* `D` Decrement

* `²f` Convert to float

* `²c` Product

* `h` Get the ASCII character of the value
